# 🏨 AHC TEAM Portal

Modern and dynamic event management platform designed for the **AHC TEAM**. This portal provides real-time event tracking, localized weather information, and a sophisticated notification system for guests and staff.

[Giriş (Türkçe)](#türkçe) | [Overview (English)](#english)

---

## 🇺🇸 English

### ✨ Key Features
- **Unified Header & Status Bar**: Real-time "LIVE" and "UPCOMING" activity tracking with premium glassmorphism design.
- **Kuşadası Weather & Clock**: Integrated live local time and real-time weather data for Kuşadası.
- **Advanced Notification Hub**: Customizable reminder system with PWA support and smart persistence.
- **7-Language Support**: Fully localized in Turkish, English, German, Russian, Bulgarian, French, and Polish.
- **Progressive Web App (PWA)**: Installable on mobile and desktop with offline support.
- **Admin Dashboard**: Full control over daily, evening, and mini club programs.

### 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5, CSS3 (Custom Glassmorphism), JavaScript (ES6+)
- **Icons**: Phosphor Icons
- **APIs**: Open-Meteo API (Weather)
- **Deployment**: Optimized for GitHub Pages

## 🚀 Setup & Deployment
1. Clone the repository:
   ```bash
   git clone https://github.com/mrdjkeremcan/ahc-portal.git
   ```
2. Open `index.html` in any modern browser.
3. For PWA features, serve the project via HTTPS (e.g., GitHub Pages).
